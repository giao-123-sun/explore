## 一、市场全景

### 市场规模

- **2024年**：中国 AI 数字员工解决方案市场规模达 **65亿元**，2020-2024 CAGR 37.5%

- **企业级 AI Agent 市场**：2023-2027 CAGR 120%，2027年将达 655亿元
- **RPA+AI 解决方案**：2023年 24.7亿，预计2026年突破 70亿元

### 渗透率

- 智能客服渗透率 >70%（最成熟场景）
- 数据分析场景渗透率 60%
- 中国 RPA 行业渗透率仅 13%，增长空间巨大
- **预测**：2026年 60% 的企业将部署 AI Agent

### 应用行业分布（2026预测）

| 行业 | 占比 |
|------|------|
| 金融服务 | 25.0% |
| 零售 | 16.9% |
| 政务 | 18.2% |

### 竞争格局

1. **协同办公平台**：钉钉、飞书、企业微信（流量入口型）

3. **大模型厂商**：智谱、字节/豆包、百度、腾讯混元（基础设施层）

### 2.1 飞书 aily（字节跳动）

**最新动态（2026年3月19日发布会）**

- **aily 升级为内置 Agent**：从对话助手升级为"企业版 OpenClaw"，原生集成飞书全部数据（群聊、文档、日程、多维表格、任务）
- **双版本策略**：
  - 普通版：飞书聊天框内的日常助手
  - 专业版：独立 Web/移动端工作台，支持代码执行、文档生成、数据分析
- **MCP 协议支持**：模型可自主规划调用工具，官方+第三方 MCP 市场，即配即用
- **妙搭 Agent**：升级为原生 Code Agent，支持自动扩展权限
- **多维表格 Agent**：数据可视化 + 智能分析 + 报表生成

**核心优势**
- 与飞书生态深度打通，上下文理解能力强（访问所有飞书数据）
- 多模型支持（豆包 + DeepSeek 等），不锁定单一模型
- MCP 协议开放性强，第三方扩展生态丰富
- 权限体系严格，与用户本人权限一致
- 多维表格月活即将突破 1000万

**真实用户反馈**
- 周报场景："帮我写这周的周报"，能拉出完整周报，80%框架搭好，用户只需20%时间润色
- 海底捞案例：10万一线服务员通过 AI 助手"大明白"每天服务6000+人，问题解答率 87%
- 用户评价："aily 绝不是PPT画饼，真的打通了Workflow"
- 商务部案例：妙搭自动化合同填写，将29个字段的手动处理变为智能化

**劣势**
- 飞书整体市场份额远小于钉钉，主要在科技/互联网企业
- 国际版 Lark 和国内飞书产品存在差异
- 对非飞书用户没有独立价值

**定价**
- 飞书基础版免费
- AI 高级功能需要飞书企业版/旗舰版订阅（具体价格需联系销售）
- aily 智能体开发平台对飞书用户开放

### 2.2 钉钉 AI 助理（阿里巴巴）

**产品形态**

钉钉 AI 助理是生长在企业 IM 中的"数字员工"，深度融合通义大模型，与考勤、审批、文档等系统天然打通。数字员工可进入企业通讯录，取得对应职务权限，参与组织协同和业务流程。

**关键数据**
- 截至2025年5月：创建的 AI 助理总数约 50万个
- AI 助理市场（Agent Store）上架超 700个应用
- AI 表格已有超 30万家企业使用
- 永升物业案例：人效提升5倍，每年省300万
- 菜鸟案例：80%答疑交给 AI，实现24小时全球服务
- 百丽案例：AI 练货，掀起零售门店人效革命

**企业十大 AI 场景（2025世界人工智能大会发布）**
1. 语音转文字 + IM 消息速读/摘要/总结
2. 消息翻译
3. AI 表格
4. 智能财务数据洞察
5. 员工关怀
6. 合同审查摘要
7. 会议记录
8. 文档内容纠错
9. 视频会议 AI 实时字幕
10. 业务流程自动化

**真实用户评测（来自少数派/人人都是产品经理）**

优点：
- 响应速度快，系统稳定性高
- 支持多 Agent 协同
- 拟人操作功能：用户演示一遍操作流程，AI 助理即可学习并执行
- 标题创作"紧贴核心事实，具备可用性"
- 好物种草文案讲清卖点，具备场景化推介能力

缺点：
- **事实准确性差**：生成内容存在基本事实错误（如华为 Pura70 事件生成了"衣物护理"错误内容）
- **无确认机制**：创建日程/待办不找用户确认，不支持修改
- **PC端入口不醒目**：图标缺中文提示，用户难感知
- **第三方应用质量参差**：Agent Store 应用表现"难言满意"
- **偶发技术问题**：使用中有报错和强退

**定价**

| 版本 | 价格 | 适合 |
|------|------|------|
| 免费版 | 0 | 5人以下小微团队 |
| 创业版 | 980元/年 | 初创企业 |
| 专业版 | 9,800元/年 | 中大型企业 |
| 专属版 | 28,000元/年起 | 大型企业定制 |
| AI 模型调用（专业版） | +10,000元/年 | 20万次大模型调用 |
| AI 模型调用（专属版） | +20,000元/年 | 45万次大模型调用 |
| 个人版 365会员 | 19元/月（优惠价） | 个人用户 |

### 2.3 企业微信 + 腾讯混元

**当前状态**

企业微信已上线"智能机器人"和"智能表格-客户跟进总结"功能，支持 DeepSeek-R1 和混元大模型。企业可简单配置创建自己的 AI 机器人。

**核心优势**
- **14亿月活**的微信生态是最大护城河
- 企业微信与微信打通，天然触达 C 端用户
- SCRM（社交客户关系管理）场景无可替代
- 混元大模型支持智能知识库，自动归档会议记录和客户沟通日志

**重磅未来产品（微信 AI Agent 项目）**

这是腾讯的"绝密项目"：
- **项目负责人**：微信技术总监周颢，直接向张小龙汇报
- **启动时间**：2025年上半年
- **计划时间线**：2026年中开始灰度测试，Q3逐步放量
- **核心设计**：直接打通微信数百万小程序，Agent 可执行打车、点外卖、购物、购票、支付等操作
- **技术路线**：测试了智谱、阿里、DeepSeek 以及微信自研小模型
- **潜在影响**：如果成功，将成为中国覆盖面最广的 AI Agent 产品

**AI 生态投资**
- 2025年投入 180亿元 AI 新品研发
- 2026年计划至少翻倍

**劣势**
- AI 功能相对钉钉和飞书**起步最晚**
- 企业微信本身的协同办公能力弱于钉钉/飞书
- 微信 AI Agent 仍在开发中，尚未面世
- 企业微信更偏 SCRM，内部协同不是强项

## 三、扣子 Coze（字节跳动 AI Agent 开发平台）

### 产品演进

1. **2023年底**：Bot Studio 无代码聊天机器人（初期留存不理想）
2. **2024-2025**：转向工作流编排（工作流调用量持续攀升，用户稳定）
3. **2025年4月**：推出扣子空间（Coze Space），AI 协同办公平台
4. **2025年7月**：**开源 Coze Studio + Coze Loop**（Apache 2.0 许可）
5. **2026年1月**：**Coze 2.0** 发布，四大核心能力：
   - Agent Skills（技能系统）
   - Agent Plan（规划能力）
   - Agent Office（办公协同）
   - Agent Coding（代码生成）

### 核心定位

从"AI 工具"升级为"主动协作伙伴"，定位**"职场AI，就用扣子"**。目标用户是有商业化目的的"严肃开发者"。

### 定价

| 版本 | 价格 |
|------|------|
| 基础版 | 免费（有额度限制） |
| 个人进阶版 | 9.9元/月 |
| 团队版 | 178元/月 |
| 企业版 | 4,980元/月 |

### 优势
- 背靠字节生态（豆包模型 + 火山引擎基础设施）
- 开源策略降低开发者门槛
- 丰富的插件系统、记忆库、工作流
- 操作界面设计合理，上手容易
- 支持多平台一键部署
- Coze 2.0 的 Agent RL + 模型后训练能力

### 劣势
- **定制性受限**：封装程度高，知识库操作灵活性不足
- **输出质量不稳定**：用户反馈回答准确性和完整性不够
- **开放性不足**：国内版主要集成本土模型，缺乏对 OpenAI/Claude 等的支持
- **企业级功能待完善**：专业深度有限
- **从工具到留存的挑战**：初期用户留存不理想，转向工作流后改善

### 与飞书的关系

扣子和飞书 aily 同属字节跳动体系，但定位不同：
- **飞书 aily**：企业内部协同的 Agent 平台，深度绑定飞书生态
- **扣子**：开放的 AI Agent 开发平台，面向更广泛的开发者和企业

## 六、Manus（已出海的中国 AI Agent）

- **背景**：2025年3月由中国团队 Monica 发布，号称"全球首个通用型 AI Agent"
- **能力**：独立完成任务，调用工具/写代码/操作网页
- **融资**：7500万美元 C轮，估值 5亿美元（Benchmark领投）
- **出海**：2025年7月迁至新加坡/东京/旧金山，**不再在中国提供服务**
- **用户评价**：演示效果惊艳，但实际测试结果参差；部分用户报告界面混乱；程序员表示测试结果不理想
- **离开中国的原因**：AI Agent 全球化面临合规与代码审查挑战

## 七、关键洞察与判断

### 7.1 三大平台的 AI 竞争格局

**钉钉**：用户基数最大（7亿），AI Agent 落地案例最多，但 AI 质量（准确性/可靠性）是短板。适合已经在钉钉生态内的企业，不适合追求 AI 前沿体验的团队。

**飞书**：AI 产品成熟度最高，aily + MCP 的开放架构最有想象力。但市场份额受限，主要在科技/互联网圈子。如果你的团队已经在用飞书，aily 是目前体验最好的企业级 Agent。

**企业微信**：最大的牌是微信 14亿月活 + 小程序生态。微信 AI Agent 项目如果成功落地（预计2026下半年），将是降维打击。但目前 AI 功能最弱，处于追赶状态。

### 7.2 RPA 厂商的困境与机遇

传统 RPA 厂商（实在智能、来也科技等）正被迫向 AI Agent 转型。IDC 预测到2026年市场将分化为"流程工匠"和"智能代理服务商"两大阵营，后者市场份额将突破 60%。

来也科技是唯一入选 Everest Group 全球 Agentic AI 评估的中国公司，转型相对成功。实在智能在中小企业市场口碑不错，但面临大厂平台的挤压。

### 7.3 定价趋势

整体呈现**从补贴向价值回归**的趋势：
- Coze 从免费走向收费（企业版 4,980元/月）
- 钉钉 AI 调用单独收费（专业版+10,000元/年）
- 大模型 API 价格持续下降（腾讯云 0.8元/百万 Token 输入）
- 数字员工综合月成本 200-3,000元，约为真人的 1/10

### 7.4 2026年关键趋势

1. **多智能体协同**：从单个 Agent 到多 Agent 编排将成为主流
2. **MCP 协议扩散**：飞书率先支持，预计钉钉/企业微信将跟进
3. **微信 AI Agent 是最大变量**：如果2026下半年成功落地，将重塑整个市场
4. **认知型 Agent 覆盖率**：预计覆盖 70% 企业复杂决策场景
5. **开源趋势**：Coze 开源、AutoGLM 开源，降低企业自建门槛
6. **出海压力**：Manus 被迫出海说明国内合规环境对 Agent 类产品有约束

### 7.5 选型建议

| 企业类型 | 推荐方案 |
|---------|---------|
| 科技/互联网团队 | 飞书 aily + Coze |
| 制造/建筑/连锁 | 钉钉 AI 助理 |
| 零售/消费/私域运营 | 企业微信 + SCRM |
